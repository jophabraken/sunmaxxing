# ZOEY — 4-week content calendar

Rolling 4-week plan. ZOEY updates this every Monday: bump dates, add newly discovered keywords, move shipped rows to the "shipped" log at the bottom.

Rules:
- **Max 2 new pages per day** (hard cap in the playbook). Usually 1 EN + 1 DE pair.
- **P0 first, then P1**, keep sun-angle long-tails for Fridays (low KD = fast wins = morale).
- **Seasonal content** is date-anchored — do not push it. If May Day prep is late, drop it; don't ship it on May 3.
- **Every EN page gets a hreflang-paired DE page within 7 days**, or we lose the cluster effect.

Legend: `EN+DE` = ship both on the same day. `EN` / `DE` = ship one, its pair is scheduled within the same week.
